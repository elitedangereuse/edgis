## API
### Development Setup
Fetch the EDTS submodule once after cloning:

```bash
git submodule update --init --recursive
```

Then set up the virtual environment and dependencies:

```bash
cd api
python3 -m venv .direnv
. .direnv/bin/activate
pip install -r requirements.txt
```
